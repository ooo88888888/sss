符号与参数（Word 版）

说明：本文件可直接用 Microsoft Word 打开，编辑后“另存为 → Word 文档（.docx）”。数学表达采用线性输入格式，便于复制到 Word 公式框自动渲染。

一、集合与索引

  • 天：t ∈ T = {1,…,365}
  • 阶段：k ∈ K = {MA, PR, LA, RE, FA}
  • 批次：b ∈ B

二、容量常数

cMA=14, cPR=8, cLA=6, cRE=14, cFA=14, cRAM=4

三、规模变量

  • M（基础母羊数）
  • R（种公羊数），约束：R ≥ ⌈M/50⌉

四、决策变量（问题二/三）

  • xb（第 b 批母羊数）
  • sb（开配日）
  • yk(t)（阶段 k 在日 t 占用栏数）
  • 随机场景：yk(t, ω), uk(t, ω), vk(t, ω)

五、常用公式（可复制到 Word 公式框）

  • 年出栏（仅统计当年内完成育肥者）：
    N = ∑b∈B 2·xb · 𝟙{ qb ∈ T }, qb = wb + τFA
    提示：将直线公式复制到 Word 的“插入 → 公式”中，使用 \tau, \lceil · \rceil 等输入即可自动渲染。
  • 日容量与总约束：
    ∑k∈K yk(t) + yRAM(t) ≤ S, ∀ t ∈ T
  • 向上取整（示例）：
    yFA(t) ≥ ⌈ OFA(t) / cFA ⌉

七、可直接粘贴到“公式框”的线性输入示例

  • \alpha_k = \tau_k / \tau_{CY},\quad E_k = M \alpha_k
  • \lambda = 2 / \tau_{CY},\quad E_{FA} = \lambda \,\tau_{FA}\, M
  • p_{FA}(t) \ge \lceil O_{FA}(t)/c_{FA} \rceil,\quad \sum_{k\in K} y_k(t) + y_{RAM}(t) \le S
  • N=\sum_{b\in B} 2x_b \cdot \mathbb{1}\{q_b \in T\},\ q_b=w_b+\tau_{FA}
